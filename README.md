# Technical-Test
Technical-Test

1. Pastikan sudah install jvm di device
2. Pull dan open source code dari github "https://github.com/zikrirahman26/Technical-Test.git"
3. Run source code dengan port default : 8080
4. Untuk penggunaan API nya bisa di coba dengan endpoint berikut:
    # fibonacci
    --> http://127.0.0.1:8080/api/fibonacci/{n}
    Keterangan : n = index
    # palindrom
    --> http://127.0.0.1:8080/api/palindrome/{s}
    Keterangan : s = string
